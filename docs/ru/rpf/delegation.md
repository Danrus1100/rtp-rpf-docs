# Система делегирования

## О стратегиях выборки

в **RPF** есть 3 стратегии выборки: `rpf:v1`, `minecraft:vanilla` и `rpf:experimental`. Вы можете переключаться между ними, написав в чат `/rpf resolver set {id}`, а узнать текущий - `/rpf resolver`

### rpf:v1

Просматривает модели из всех наборов сверху вниз, прерывая цикл, когда модель не делегировала, рендеря её

### rpf:experimental

Сначала тестирует все модели, а потом, исходя из очков, которые модель набрала (они набираются и вычитаются в зависимости от глубины модели и их типа), выбирает модель с самым большим баллом и рендерит её

::: warning Обратите внимание
Стратегия названа "Экспериментальной", потому что она **НЕ** стабильна! В будущих обновлениях она также может быть переименована
:::

### minecraft:vanilla

Стандартная стратегия выборки, написанная Mojang
