# Система делегирования

## О стратегиях выборки

в **RPF** есть 3 стратегии выборки: `rpf:v1`, `minecraft:vanilla` и `rpf:experemental`. Вы можете переключаться между ними, написав в чат `/rpf resolver set {id}`, и узнать текущую выборку при помощи команды `/rpf resolver`

### rpf:v1

Просматривает модели из всех наборов сверху вниз, прерывая цикл когда модель не делегировала, рендеря её

### rpf:experemental

Сначала тестирует все модели, а потом, судя по очкам, которые модель набрала (они набираются и вычитаются в зависимости от глубины модели и их типа), выбирает модель с самым большим баллом и рендерит её

::: warning Обратите внимание
Стратегия названа "Экспериментальной", потому что она **НЕСТАБИЛЬНА!** В будущих обновлениях она также может быть переименована
:::

### minecraft:vanilla

Стандартная статегия выборки, написанная Mojang
