# Свойства

**RPT** добавляет новые свойства для `select` и `condition` моделей:

## Select свойства
 1. `rpt:biome` - в каком биоме находится держатель предмета
 2. `rpt:weather` - какая погода в мире. может быть `none` (если предмет рендериться вне мира), `clear` и `rain`. ~~`thunder` технически тоже есть, но это нельзя отследить в ванильном клиенте игры~~
 3. `rpt:difficulty` - какая сложность игры выставленная. может быть `none` (если предмет рендериться вне мира), `hardcore`, `peaceful`, `easy`, `normal` и `hard`

## Condition свойства
 1. `in_fluid` - находится ли держатель предмета в жидкости
 2. `entity_flag` - принимает `flag`, который может быть: 
     - `on_fire` - в огне ли сущность, 
     - `sneaking` - крадётся ли сущность, 
     - `sprinting` - бежит ли сущность, 
     - `swimming` - плывёт ли сущность, 
     - `gliding` - летает ли сущность на элитрах,
     - `climbing` - забирается ли сущность по лестнице, 
     - `is_player` - игрок ли сущность.
