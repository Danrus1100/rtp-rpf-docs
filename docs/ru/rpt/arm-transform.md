# Вращение рук

### Ванильные вращения
Вы можете задавать вращение руки для предмета, который игрок держит в руке! Это делается "оборачиванием" вашей модели в модель `rpt:arm_transform`:

```json{3-4}
{
    "model": {
        "type": "rpt:arm_transform",
        "transform": "item",
        "model": {
            "type": "model",
            "model": "item/diamond_pickaxe"
        }
    }
}
```

В примере выше мы указали вращение руки, которое используется в ванильной игре, когда игрок держит предмет. Ниже список ванильных вращений:
 - `empty` - игрок ничего не держит
 - `item` - игрок держит предмет, не вызывающий вращений ниже
 - `block` - игрок поднимает щит
 - `bow_and_arrow`- игрок натягивает тетиву лука
 - `throw_trident`- игрок замахивается трезубцем
 - `crossbow_charge`- игрок заряжает арбалет
 - `crossbow_hold` - игрок держит заряженный арбалет
 - `spyglass` - игрок смотрит в подзорную трубу
 - `toot_horn` - игрок дует в козий рог
 - `brush` - игрок машет кисточкой.
 - `spear` - игрок использует копьё как кинетическое оружие (Для 1.21.1 и выше)

Помимо ванильных вращений, вы можете задавать свои собственные, либо изменять некоторые настройки ванильных: 

### Создание собственного вращения

```json
{
    "model": {
        "type": "rpt:arm_transform",
        "transform": {
            // вращение по осям
            "x": 0, 
            "y": 0,
            "z": 0,
            "bob": true, // применять ли анимацию "дыхания"
            "swing": true, // применять ли анимацию качания рук при ходьбе
            "attack": true, // применять ли анимацию атаки
            "type": "item" // тип анимаций из ванильных
        },
        "opposite" : { ... }, // Трансформации противоположной руки, если другой предмет с rpt:arm_transform её не перебивает
        "model": { ... }
    }
}
```
::: info Примечание
Если вы в записи выше укажете `type`, то вращения по осям **НЕ** будут применяться!
:::

Помимо чисел, оси поддерживают выражения (подробнее об этом в [`Математические Выражения`](expressions.md)): 

```json
{
    "model": {
        "type": "rpt:arm_transform",
        "transform": {
            "x": "sin(time*0.75) * 45 - 90", // [!code focus] Игрок будет качать рукой перед лицом
            "swing": false
        },
        "model": {
            "type": "model",
            "model": "item/bell"
        }
    }
}
```
RPT так же предоставляет некоторые переменные, которые можно использовать:

 - `time` - игровое время в тиках
 - `hx/hy/hz` - поворот головы по осям
 - `useArm` - рука, которая использует предмет: 1 если главная, и -1 если вторичная
 - `swingArm` - рука, которой производится удар: 1 если правая, и -1 если левая
 - `useTick` - сколько тиков применения предмета
 - `holdArm` - рука, которая держит предмет: 1 если правая, и -1 если левая

::: warning Обратите внимание
Переменные, о которых говорится в [`Математические Выражения`](expressions.md) не будут работать в `rpt:arm_transform`! 
:::
